where Self: 'async_trait, 'life0: 'async_trait,

Intercept an outgoing HTTP request.

If this method returns [InterceptedResponse::Continue], the (possibly updated) request will be passed on to the default outgoing request handler.

If this method returns [InterceptedResponse::Intercepted], the inner result will be returned as the result of the request, bypassing the default handler. The request will also be dropped immediately.
