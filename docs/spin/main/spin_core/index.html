Spin core execution engine

This crate provides low-level Wasm and WASI functionality required by Spin. Most of this functionality consists of wrappers around wasmtime and wasmtime_wasi that narrows the flexibility of wasmtime to the set of features used by Spin (such as only supporting wasmtime’s async calling style).
